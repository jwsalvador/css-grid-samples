CSS Grid

CSS Grid is best use to control two-dimensional layouts (rows & columns) 

              
      .wrapper {
          display: grid;
      }

Example 1

Defining columns

    .wrapper {
        display: grid;
        grid-template-columns: 200px 200px 200px;
        grid-gap: 5px;
    }

In this example, I've introduced grid-template-columns and grid-gap.

You can specify the number of columns by using grid-template-columns. The sample above specifies that wrapper should have 3 columns of 200px each.

To make it shorter, you can use repeat(n, x) value. n being the repeat number, x being the size of the column. 

          
    .wrapper {
        display: grid;
        grid-template-columns: repeat(3, 200px);
        grid-gap: 5px;
    }

Now that columns have been defined, grid-gap allows you to evenly space all grid-items accordingly.

    .example2 .wrapper {
        display: grid;
        grid-template-columns: repeat(4, 50px);

    .example2 .wrapper .item2 {
        grid-column: 2 / 5;
    }
    .example2 .wrapper .item3 {
        grid-column: 1 / 3;
        grid-row: 2 / 4;
    }
    .example2 .wrapper .item6 {
        grid-column-start: 4;
        grid-column-end: 5;
        grid-row-start: 2;
        grid-row-end: 4;
    }
    .example2 .wrapper .item8 {
        grid-column-start: 1;
        grid-column-end: 5;
        grid-row-start: 4;
        grid-row-end: 5;
    }

This demonstrates row and column positioning on a specified grid line using grid-column-start, grid-column-end, grid-row-start and grid-row-end.

For .item6 example above, grid-column-start: 4; grid-column-end: 5; tells the grid item to start displaying on column number 4 and should end in column number 5. grid-row-start: 2; grid-row-end: 4; on the other hand, tells the grid item to start displaying on row number 2 and should end in row number 4.

Short hand method for the example above would be, grid-column: 4 / 5; grid-row: 2 / 4;

The fr unit

As the name Fraction unit suggests, this unit represents a fraction of the available space in the grid container.

Say we've got a whole pizza, which represent 1 whole fraction (grid-template-columns: 1fr;) or 100%.

1fr

Now, if we slice it in half, it will now give us 2 columns of 1fr each, grid-template-columns: 1fr 1fr; which is divided equally. Now my wife wants more share on the pizza, grid-template-columns: 2fr 1fr;, which means that she'll get two-thirds of the whole pizza, and I'll get only one-third of the pizza.

grid-template-areas

This is a pretty neat way of positioning certain grid items on the grid container.

On the example above, we've got 3 columns, and 4 rows as specified by:

          
  grid-template-columns: 1fr 2fr 1fr;
  grid-template-rows: auto 60px 150px auto;

grid-template-areas allows you to position grid items just like you're describing it in words. 

          
  grid-template-areas: 
  "nav nav nav"
  "header header right"
  "left main right"
  "footer footer footer";

On the first row, we've got 3 navs, which denotes the whole first row would be occupied by nav grid area.

For the second row, 2 columns are occuppied by the header grid item, and one column occupied by the right side bar.

Third row gives you a bit of flexibility, where the 1st column is left area, 2nd column is the main area, finally 3rd column is the right area, which also extends to the 2nd row.

Fourth row is the same as the 3 navs above, occupying the whole row.

grid-area

grid-template-areas alone is not sufficient enough to get you your desired position, you need to assign the name you specified in grid-template-areas to the grid item's grid-area.

The nav name inside grid-template-areas needs to be assigned to a grid item, in the case above, div.nav is suitable for the "nav" area, so we specify: 

            
  .nav {
    grid-area: nav;
  }

Same goes with the other grid areas specified.
